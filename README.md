운칠기삼 - 점집과 고객 연결 웹 사이트

1. 구현 목표
- 사용자들에게 원하는 유형의 점집 소개
- 제휴사들의 투명한 정보 공개로 인한 경쟁심화, 사용자에게 양질의 컨텐츠 제공
- 제휴사에게 마케팅, 고객과 스케줄 관리, 자동화된 수익 인프라 제공

2. 개발 환경
  - 프로젝트 환경
    - spring MVC
    - Java 11
    - Tomcat v9.0
    - Github
  - 라이브러리

    - MyBatis (v3.5.11)

    - spring-jdbc 5.3.27
    - spring-websocket 5.3.27
    - ORACLE clode
